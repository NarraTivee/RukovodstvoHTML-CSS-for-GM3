## 🚀 Установка

### Простой способ (рекомендуется)

1. Скачайте или клонируйте репозиторий
2. Откройте файл `index.html` в браузере
3. Начните обучение!

### Для разработки

1. Клонируйте репозиторий:
```bash
git clone https://github.com/yourusername/modern-html-css-tutorial.git
cd modern-html-css-tutorial

2. Откройте проект в вашем редакторе кода
3. Запустите локальный сервер (опционально):
# Используя Python
python -m http.server 8000
# Или используя Node.js
npx serve .

4. Откройте http://localhost:8000 в браузере


🎮 Использование

Навигация
Используйте sidebar для навигации по разделам
На мобильных устройствах нажмите кнопку меню в верхнем левом углу
Используйте кнопку "Наверх" для быстрого возврата к началу

Демонстрации
Каждый раздел содержит интерактивные демонстрации:
Нажимайте кнопки для изменения свойств
Наводите курсор на элементы для preview эффектов
Используйте контролы для управления анимациями
Прогресс обучения
Прогресс-бар в sidebar показывает ваш прогресс
Прогресс автоматически сохраняется в localStorage
Вы можете продолжить с того места, где остановились
