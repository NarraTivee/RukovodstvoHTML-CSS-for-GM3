Структура HTML документа

Базовая структура HTML5 документа

Каждый HTML5 документ имеет определенную структуру, включающую важные элементы. Понимание этой структуры является основой веб-разработки.

<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Заголовок страницы</title>
    <!-- Внешний CSS -->
    <link rel="stylesheet" href="styles.css">
    <!-- Внутренний CSS -->
    <style>
        body { font-family: Arial, sans-serif; }
    </style>
</head>
<body>
    <h1>Мой первый заголовок</h1>
    <p>Мой первый параграф.</p>
    <script src="script.js"></script>
</body>
</html>

<!DOCTYPE html>

Объявление типа документа - Определяет документ как HTML5. Это первая строка в любом HTML5 документе.

<html> тег

Корневой элемент - Контейнер для всех других HTML элементов (кроме !DOCTYPE). Атрибут lang определяет язык.

<head> секция

Метаданные документа - Содержит метаинформацию, заголовок, ссылки на стили и скрипты.

<title> тег

Заголовок страницы - Определяет заголовок веб-страницы в браузере и поисковых системах.

<meta> теги

Метаданные - Предоставляют информацию о HTML документе:

  • charset="UTF-8" - Кодировка символов
  • viewport - Адаптивность на мобильных
  • description - Описание для SEO
  • keywords - Ключевые слова

<body> тег

Тело документа - Содержит весь видимый контент страницы: текст, изображения, ссылки и т.д.

Основы и стилизация CSS

Каскадные таблицы стилей (CSS)

CSS описывает как HTML элементы должны отображаться на экране, бумаге или других медиа.

Способы подключения CSS к HTML файлу:

1. Внешний CSS

Самый распространенный - Ссылка на внешний .css файл.

<link rel="stylesheet" href="styles.css">

2. Внутренний CSS

Использование элемента <style> в секции <head>.

<style>
    body { background-color: lightblue; }
</style>

3. Встроенный CSS

Использование атрибута style в HTML элементах.

<h1 style="color:blue;">Заголовок</h1>

CSS Box Model

Каждый элемент в веб-дизайне - прямоугольный блок. Box Model описывает как эти блоки размеряются.

Margin
Border
Padding
Content
/* Пример CSS Box Model */

/* Основные свойства Box Model */
.box-element {
    /* Content */
    width: 300px;
    height: 200px;
    
    /* Padding - внутренний отступ */
    padding: 20px; /* Все стороны */
    padding: 20px 30px; /* Вертикально и горизонтально */
    padding: 10px 20px 30px 40px; /* Верх, право, низ, лево */
    
    /* Border - граница */
    border: 3px solid #333; /* Толщина, стиль, цвет */
    border-width: 2px;
    border-style: solid;
    border-color: blue;
    
    /* Margin - внешний отступ */
    margin: 15px; /* Все стороны */
    margin: 10px auto; /* Центрирование по горизонтали */
    margin-top: 20px;
    margin-right: 30px;
    margin-bottom: 20px;
    margin-left: 30px;
}

/* Пример HTML */
<div class="box-element">
    Содержимое элемента
</div>

/* Рассчет полного размера элемента:
   Общая ширина = width + padding-left + padding-right + border-left + border-right + margin-left + margin-right
   Общая высота = height + padding-top + padding-bottom + border-top + border-bottom + margin-top + margin-bottom
*/

    });
});

Flexbox верстка

Flexbox - одномерный метод верстки для расположения элементов в рядах или колонках.

Направление: row Выравнивание: flex-start Перенос: nowrap

/* Основные свойства Flexbox */

/* 1. Создание flex-контейнера */
.flex-container {
    display: flex; /* Активация Flexbox */
}

/* 2. Направление элементов (flex-direction) */
.flex-container {
    flex-direction: row; /* По умолчанию - в строку */
    flex-direction: row-reverse; /* В строку в обратном порядке */
    flex-direction: column; /* В колонку */
    flex-direction: column-reverse; /* В колонку в обратном порядке */
}

/* 3. Выравнивание по главной оси (justify-content) */
.flex-container {
    justify-content: flex-start; /* По умолчанию - в начале */
    justify-content: flex-end; /* В конце */
    justify-content: center; /* По центру */
    justify-content: space-between; /* Равномерно с отступами между */
    justify-content: space-around; /* Равномерно с отступами вокруг */
    justify-content: space-evenly; /* Равномерно с равными отступами */
}

/* 4. Перенос элементов (flex-wrap) */
.flex-container {
    flex-wrap: nowrap; /* По умолчанию - без переноса */
    flex-wrap: wrap; /* С переносом на новую строку */
    flex-wrap: wrap-reverse; /* С переносом в обратном порядке */
}

/* 5. Выравнивание по поперечной оси (align-items) */
.flex-container {
    align-items: stretch; /* По умолчанию - растягивание */
    align-items: flex-start; /* В начале */
    align-items: flex-end; /* В конце */
    align-items: center; /* По центру */
    align-items: baseline; /* По базовой линии */
}

/* Пример HTML */
<div class="flex-container">
    <div class="flex-item">Элемент 1</div>
    <div class="flex-item">Элемент 2</div>
    <div class="flex-item">Элемент 3</div>
</div>

/* JavaScript для изменения flex-свойств */
document.getElementById('flexDirectionBtn').addEventListener('click', function() {
    const container = document.getElementById('flexDemoContainer');
    const currentDirection = container.style.flexDirection || 'row';
    container.style.flexDirection = currentDirection === 'row' ? 'column' : 'row';
});
